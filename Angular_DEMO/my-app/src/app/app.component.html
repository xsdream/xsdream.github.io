指令

>

全局安装cli

    npm install -g @angular/cli

创建新项目

    ng new my-app

打开server服务器

   ng serve --open

创建新的组件

   ng g c heroes(ng generate component name)

创建服务

   ng g s hero(ng generate service hero)

将服务交给组件

   ng generate service hero --module=app (交给app)



概念

元数据：

   应用程序的各个部分组合到一起，以及该应用需要哪些其它文件和库

主从主键(父子组件属性传递)

   使用@Input 可以在标签中传值 [给子的]="父的"



代码

两个大括号：

   可以将组件的值绑定到HTML中

[(ngModel)]="hero.name"：

   双向绑定

(click)="onSelect(hero)

   click事件绑定

*ngIf=""

   判断显示

*ngFor="let hero of heros"

   循环显示

[class.selected]="hero === selectedHero"

   动态增加class,注意后面是判断条件



管道：将数组转换后写入到HTML标签中使用

birthday | date

   组件的 birthday 值通过管道操作符( | )流动到 右侧的Date 管道函数中
